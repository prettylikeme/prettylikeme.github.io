Vue 基本知识点

2018-07-19
  • Html
  • Vue.js

Vue 是一个渐进式的 JavaScript 框架，采用了MVVM模型。

特点

  1. 解耦视图和数据
  2. 可复用的组件
  3. 前端路由技术
  4. 状态管理
  5. 虚拟DOM

插值语法

  1. mustache 语法: {{}}` 1. `v-once`: 只更新一次 1. `v-html`: 将文本作为 `html` 插入 1. `v-text`: 相比 `mustache` 语法，会覆盖标签内文本 1. `v-pre`: 标签内文本按原始值显示 1. `v-cloak`: 斗篷， `vue` 解析前属性存在，解析后删除该属性 ## 动态绑定属性 v-bind 1. 绑定基本属性 1. 绑定 `class` 1. 对象语法: `key:className value:boolean` 1. 数组语法: 每个元素代表一个 `class` 1. 绑定 `style` 1. 对象语法: `key:属性名 value: 属性值` 1. 数组语法: 元素为键值对 Vue 中的数据只有在一开始初始化的变量才会被响应式监控。 如果要监控新的值或者删除监控，需使用 Vue 的方法。 ## 计算属性 computed 1. 相比 `methods` ， `computed` 有缓存 1. 有 `setter` 和 `getter` 两个方法，一般不使用 `setter` 方法，故默认函数是 `getter` 方法。 ## 过滤器 filters 使用方法：在属性后面，通过管道符连接。 `{{ price | showPrice }}

v-on

  1. 事件对象
    1. 绑定的函数有一个或多个参数时，调用时省略参数列表，将把 event 事件对象作为第一个参数传递给函数
    2. 绑定的函数不需要参数时，调用时可省略参数列表，实际可以理解为同上一条
    3. 绑定的函数有多个参数时，有参数列表，且需要 event 事件对象时，可使用 $event 将事件对象传递给函数
  2. 修饰符
    1. .stop ：阻止事件冒泡
    2. .prevent ：阻止默认事件
    3. .{ keyCode | keyAlias } ：监听特定按键

v-model 双向绑定

  1. 修饰符
    1. v-model.lazy : 当控件输入回车或失去焦点时，才更新数据
    2. v-model.number : 通过这种修饰符，界面到数据的值会转换为数字
    3. v-model.trim : 更新数据时会清楚输入框中前后的空格

组件化

  1. 组件中的 data 使用函数的形式进行返回。

最简单的组件用法：

通过标签 <cpn></cpn> 的形式在 vue 实例作用域内使用。

全局组件与局部组件

  • 全局组件通过 Vue.component 方法进行注册
  • 局部组件通过创建 Vue 实例时传入的 components 属性进行注册

父组件与子组件

在组件中注册的其他组件则是该组件的子组件

组件中模板的分离式写法

通过 <script type="text/x-template" id="cpn"></script> 或 <template id="cpn"></template> 来定义 template 。

通过 id 来进行关联： const cpn = Vue.component('cpn', {template: '#cpn'}) 。

组件间的通信

父组件向子组件传递

子组件使用 props 属性来接受数据，父组件在子组件实例上使用 v-bind 向子组件传递值。

子组件向父组件传递

子组件通过 $emit 发送自定义事件，父组件使用 v-on 来监听指定的事件，以此来完成通信。

插槽 slot

插槽为组件预留一个或多个接口，方便组件拓展。

  • 匿名插槽
    • 在组件模板中使用 <slot></slot> 即可定义一个插槽，在组件标签内填入的内容即会替换到所有匿名插槽所在位置。
  • 插槽的默认值
    • 在定义插槽时，可以在内部填入内容，该内容为插槽的默认内容。
  • 具名插槽
    • 插槽也可以有 name 属性，使用时也需要在组件内的标签上使用 slot 属性。
  • 作用域插槽
    • 变量作用域：变量属于该变量所在模板所属的组件
    • 通过在插槽替换内容中添加 slot-scope="slot" 属性，即可通过 slot.<dataname> 访问插槽上绑定的数据 <slot :<dataname>="<data>"></slot>
