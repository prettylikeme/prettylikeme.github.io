Python 3.X 中的知识点

2018-07-19
  • Python
  • Python
  • Post not found: Python/DataType 数据类型:
    • 整数、浮点数、字符串、布尔值、复数
    • 字符串前带r，其内容不被转义
    • 字符串、列表（list）、元组（tuple）、集合（set）、字典（dict）
      • 列表使用[]
      • 元组使用()，不能改变元素的值
      • 集合使用{}，不允许有重复值
      • 字典也是用{}，类似map，写法类似json格式的数据
      • 一些方法
        • 下标: [pos]取pos位置的元素
        • 切片: [beg:end]取[beg, end)之间的元素，beg、end均可省略，为负数时是倒数第x个数
        • 按步长切片: [beg:end:step]，step为步长
          • step>0, 则表示从左向右进行切片。此时，beg必须小于end才有结果，否则为空。
          • step<0, 则表示从右向左进行切片。此时，beg必须大于end才有结果，否则为空。相当于beg和end颠倒位置

  • 运算符:

    • ** : 指数
    • / : 除、// : 整除
    • is 、 is not : 身份运算符
    • in 、 not in : 成员运算符
    • not 、 or 、 and : 逻辑运算符

  • 条件语句:
    if、elif、else

  • 上下文语法:
    通过with关键字指定文件对象的上下文环境并在离开上下文环境时自动释放文件资源。

  • 循环:
    while、for-in（for i in range(x)）

  • 主函数:
    if __name__ == '__main__':

  • 类class:

    • __init__(self, ...)相当于构造函数，用于定义其成员变量

    • 私有变量和方法，以__开头，但不建议使用私有变量，因为会导致子类无法访问。故将属性名以单下划线开头来表示属性是受保护的，本类之外的代码在访问这样的属性时应该要保持慎重，这种做法并不是语法上的规则，单下划线开头的属性和方法外界仍然是可以访问的，所以更多的时候它是一种暗示或隐喻

    • getter和setter方法

      • getter: 用@property来修饰
      • setter: 用@xxx.setter来修饰，xxx是属性名

    • 使用__slots__限定当前类的属性。（语法: __slots__ = ('_name', '_age', '_gender')）

    • 静态方法: 使用@staticmethod修饰

    • 类方法: 使用@classmethod修饰。类方法的第一个参数约定名为cls，它代表的是当前类相关的信息的对象（类本身也是一个对象，有的地方也称之为类的元数据对象），通过这个参数我们可以获取和类相关的信息并且可以创建出类的对象。

    • 继承: 定义类时，参数列表中为被继承的类名，可以多个继承。

    • 函数重写: 使用@override修饰

    • 抽象类: 通过abc模块的ABCMeta元类和abstractmethod包装器来达到抽象类的效果。
