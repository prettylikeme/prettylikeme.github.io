Python 中的数据类型

2018-07-19
  • Python
  • Python

整数

Python的整数没有大小限制。

浮点数

Python的浮点数也没有大小限制，但是超出一定范围就直接表示为inf（无限大）。

字符串

  • 字符串前加r可以将内容中的转义符不转义。如r'\\'输出的是\\，而不是\。
  • 字符串用三个单/双引号对包起来，可以换行输入'''text'''或"""text"""。

字符/字符串编码

  • ord()函数将字符转换为十进制整数，chr()函数将编码转换为对应字符。
  • 字符串前面加b表示的是bytes类型。bytes中每个字符都只占一个字节。
  • 字符串str类型可通过encode()方法转换为对应编码的bytes类型，如：'中文'.encode('utf-8') ==>> b'\xe4\xb8\xad\xe6\x96\x87'。
  • bytes类型可通过指定编码调用decode()方法转换为字符串str类型（如果bytes中只有一小部分无效的字节，可以传入errors='ignore'忽略错误的字节），如：b'\xe4\xb8\xad\xff'.decode('utf-8', errors='ignore') ==>> '中'。
  • len()计算str类型得到的是字符数；计算bytes类型，得到的是字节数，如：len('中文'.encode('utf-8')) ==>> 6。

保存python的源码文件时，若包含中文，在保存时，我们务必要以utf-8编码保存，并在文件开头，写上下面两行注释：
第一行告诉Linux/OS X系统，这是一个Python可执行程序，Windows系统会忽略这个注释；
第二行告诉Python解释器，按照UTF-8编码读取源代码，否则，你在源代码中写的中文输出可能会有乱码。

格式化字符串

  • 在Python中，字符串的格式化采用的方式和C语言是一致的，用%实现。如：'Hello, %s' % ('world')，只有一个%?时，括号可以省略。
  • 使用format()方法格式化，使用传入的参数依次替换字符串内的占位符{0}、{1}…。如：'Hello, {0}, 成绩提升了 {1:.1f}%'.format('小明', 17.125) ==>> 'Hello, 小明, 成绩提升了 17.1%'。

布尔值

Python中的boolean类型的取值是True或False。

空值

Python中的空值用None表示。

常量

在Python中，没有能够表示常量的变量类型，习惯上通常用全部大写的变量名表示常量，如：PI=3.1415926，但实际PI的值仍然可以被改变。

运算符

Python中的/得到的结果是浮点数，而使用//的用法则类似C/C++中的/的用法。

list

list类型可以用-1来索引，取的是倒数第一个元素，-2、-3、…依次类推，同样也不能越界。
