数组

部分遍历相关函数：

  • every ：所有元素经过回调返回值都为true，则函数返回值为 true ，否则为 false
  • some ：只要有元素经过回调返回值为true，则函数返回值为 true ，否则为 false
  • includes ：包含指定的元素时返回 true
  • map ：每个元素在回调函数中的返回值，替换原来的值
  • find ：查找元素
  • findIndex ：查找元素的索引值
  • filter ：回调函数返回值为 false 的元素将被过滤
  • reduce ：用于求值，将数组中的多个元素，通过回调计算为一个值返回

函数

函数会默认带个属性 arguments ，是数组类型

call 、 apply 和 bind

这三个函数都是用来重定义函数体内的 this 指向的，只是在用法上不一样：

  • call ： func.call(obj, arg1, arg2)
  • apply ： func.apply(obj, [arg1, arg2])
  • bind ： func.bind(obj, arg1, arg2)
