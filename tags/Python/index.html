Python 3.X 中的知识点

2018-07-19
  • Python
  • Python
  • Post not found: Python/DataType 数据类型:
    • 整数、浮点数、字符串、布尔值、复数
    • 字符串前带r，其内容不被转义
    • 字符串、列表（list）、元组（tuple）、集合（set）、字典（dict）
      • 列表使用[]
      • 元组使用()，不能改变元素的值
      • 集合使用{}，不允许有重复值
      • 字典也是用{}，类似map，写法类似json格式的数据
      • 一些方法
        • 下标: [pos]取pos位置的元素
        • 切片: [beg:end]取[beg, end)之间的元素，beg、end均可省略，为负数时是倒数第x个数
        • 按步长切片: [beg:end:step]，step为步长
          • step>0, 则表示从左向右进行切片。此时，beg必须小于end才有结果，否则为空。
          • step<0, 则表示从右向左进行切片。此时，beg必须大于end才有结果，否则为空。相当于beg和end颠倒位置

  • 运算符:

    • ** : 指数
    • / : 除、// : 整除
    • is 、 is not : 身份运算符
    • in 、 not in : 成员运算符
    • not 、 or 、 and : 逻辑运算符

  • 条件语句:
    if、elif、else

  • 上下文语法:
    通过with关键字指定文件对象的上下文环境并在离开上下文环境时自动释放文件资源。

  • 循环:
    while、for-in（for i in range(x)）

  • 主函数:
    if __name__ == '__main__':

  • 类class:

    • __init__(self, ...)相当于构造函数，用于定义其成员变量

    • 私有变量和方法，以__开头，但不建议使用私有变量，因为会导致子类无法访问。故将属性名以单下划线开头来表示属性是受保护的，本类之外的代码在访问这样的属性时应该要保持慎重，这种做法并不是语法上的规则，单下划线开头的属性和方法外界仍然是可以访问的，所以更多的时候它是一种暗示或隐喻

    • getter和setter方法

      • getter: 用@property来修饰
      • setter: 用@xxx.setter来修饰，xxx是属性名

    • 使用__slots__限定当前类的属性。（语法: __slots__ = ('_name', '_age', '_gender')）

    • 静态方法: 使用@staticmethod修饰

    • 类方法: 使用@classmethod修饰。类方法的第一个参数约定名为cls，它代表的是当前类相关的信息的对象（类本身也是一个对象，有的地方也称之为类的元数据对象），通过这个参数我们可以获取和类相关的信息并且可以创建出类的对象。

    • 继承: 定义类时，参数列表中为被继承的类名，可以多个继承。

    • 函数重写: 使用@override修饰

    • 抽象类: 通过abc模块的ABCMeta元类和abstractmethod包装器来达到抽象类的效果。

Python 中的数据类型

2018-07-19
  • Python
  • Python

整数

Python的整数没有大小限制。

浮点数

Python的浮点数也没有大小限制，但是超出一定范围就直接表示为inf（无限大）。

字符串

  • 字符串前加r可以将内容中的转义符不转义。如r'\\'输出的是\\，而不是\。
  • 字符串用三个单/双引号对包起来，可以换行输入'''text'''或"""text"""。

字符/字符串编码

  • ord()函数将字符转换为十进制整数，chr()函数将编码转换为对应字符。
  • 字符串前面加b表示的是bytes类型。bytes中每个字符都只占一个字节。
  • 字符串str类型可通过encode()方法转换为对应编码的bytes类型，如：'中文'.encode('utf-8') ==>> b'\xe4\xb8\xad\xe6\x96\x87'。
  • bytes类型可通过指定编码调用decode()方法转换为字符串str类型（如果bytes中只有一小部分无效的字节，可以传入errors='ignore'忽略错误的字节），如：b'\xe4\xb8\xad\xff'.decode('utf-8', errors='ignore') ==>> '中'。
  • len()计算str类型得到的是字符数；计算bytes类型，得到的是字节数，如：len('中文'.encode('utf-8')) ==>> 6。

保存python的源码文件时，若包含中文，在保存时，我们务必要以utf-8编码保存，并在文件开头，写上下面两行注释：
第一行告诉Linux/OS X系统，这是一个Python可执行程序，Windows系统会忽略这个注释；
第二行告诉Python解释器，按照UTF-8编码读取源代码，否则，你在源代码中写的中文输出可能会有乱码。

格式化字符串

  • 在Python中，字符串的格式化采用的方式和C语言是一致的，用%实现。如：'Hello, %s' % ('world')，只有一个%?时，括号可以省略。
  • 使用format()方法格式化，使用传入的参数依次替换字符串内的占位符{0}、{1}…。如：'Hello, {0}, 成绩提升了 {1:.1f}%'.format('小明', 17.125) ==>> 'Hello, 小明, 成绩提升了 17.1%'。

布尔值

Python中的boolean类型的取值是True或False。

空值

Python中的空值用None表示。

常量

在Python中，没有能够表示常量的变量类型，习惯上通常用全部大写的变量名表示常量，如：PI=3.1415926，但实际PI的值仍然可以被改变。

运算符

Python中的/得到的结果是浮点数，而使用//的用法则类似C/C++中的/的用法。

list

list类型可以用-1来索引，取的是倒数第一个元素，-2、-3、…依次类推，同样也不能越界。
